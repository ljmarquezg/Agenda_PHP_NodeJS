    • content_copy3.1 - Copiar archivos del proyecto en la carpeta htdocs de xampp.arrow_drop_down
      Copiar la carpeta agenda_php dentro del directorio xampp/htdocs. La ruta por defecto:

    • mode_edit3.2 - Editar la conficuración del archivo .arrow_drop_down

      Verificar los parámetros localhost, username, password del archivo conector.php contenido en la carpeta agenda_php/server

      La ruta por defecto: C:\xampp\htdocs\agenda_php\server\conector.php

      Si se configura el archivo con a información de un usuario con permisos administrativos de phpmyadmin, se puede omitir el paso siguiente ya que el sistema generará automáticamente la base de datos e insertará la información del usuario
    • import_export3.3 Importar la base de datos en phpmyadmin arrow_drop_down
      1. Abrir en el explorador phpmyadmin. localhost:PUERTO/phpmyadmin
      2. Crear la base de datos por defecto agenda_db
      3. Importar la información de la base de datos utilizando el archivo agenda_db.slq
    • code3.4 Abrir el proyecto arrow_drop_down
